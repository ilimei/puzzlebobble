# puzzlebobble

> The Tiny.js game project build by tinyjs-cli

![](https://zos.alipayobjects.com/rmsportal/nJBojwdMJfUqpCWvwyoA.png@120w)

## 命令

- `npm install`: 安装依赖
- `npm start`: 本地服务，默认端口：8017
- `npm build`: 执行编译

## 关于 Tiny.js

- `官网`: http://tinyjs.net
